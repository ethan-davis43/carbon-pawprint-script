## Project Summary

Carbon Pawprint is a lightweight web game that teaches eco awareness through travel-themed decisions. Each run asks players to pick between options that shape the journey and influence the final environmental result.

Designed for quick browser play, the experience uses endangered animal characters, a running tally of decision impact, and an end-of-run sustainability score to turn the game into a simple educational activity.

---

## Script Features

- Fast, decision-driven gameplay meant for short play sessions
- Travel choices with sustainability consequences
- Endangered animal protagonists presented in an educational context
- Tally system that records the effect of each choice
- Final sustainability score displayed when the session ends
- Web demo delivery for browser access
- Centered on eco awareness and sustainability themes
- Built as a hackathon project
